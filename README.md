# AfriTalent

**Nom :** FAYE Paul Augustin Stanislas  
**Classe :** l1CS  
**Année :** 2026

## Description
Plateforme fictive de mise en relation entre freelances tech et entreprises en Afrique.

## Technologies utilisées
- HTML5
- CSS3 (Flexbox, Grid, Bento Grid)
- Bootstrap 5
- JavaScript Vanilla
- Git & GitHub

## Fonctionnalités
- Dark mode
- Compteurs animés
- Filtrage dynamique des freelances
- Validation formulaire
- Navbar dynamique
- Animations fade-in
- Bouton retour en haut

1. Cloner le dépôt
2. Ouvrir index.html avec Live Server
